GitHub Api

Github Api ini masih dalam perkembangan devloper

Untuk Cara GitHub ini cukup gampang tinggal masukin url diatas di tambahi /github/ Link

fitur dari api ini Yaitu:p>

  • github bassic /github/{nama} <-- didalam kurung itu tulis nama github kalian seperti /github/dodokjr
  • github reppos /github/{nama}/repos <-- didalam kurung itu tulis nama github kalian dan ditambah kata repos seperti /github/dodokjr/repos

    • Di atas itu adalah Installation api github.

    akan datang terbaru / update terbaru dari mfikria untuk api github ini.

    Anime Api

    Anime Api ini masih dalam perkembangan devloper

    Untuk Cara GitHub ini cukup gampang tinggal masukin url diatas di tambahi /anime Link secara index api ini langsung menampilkan halaman home.

    fitur dari api ini Yaitu:

  • Anime Home /anime secara default /anime ini udah home base dari api
  • Anime Seacrh /anime/{query} <-- didalam kurung tulis query anime yang kamu cari seperti /anime/naruto

    • Di atas itu adalah Installation api anime.

    akan datang terbaru / update terbaru dari mfikria untuk api anime ini.
